# tfe
Terraform API utility

## Usage
```
tfe v0.3.0
Terraform Cloud/Enterprise CLI utility. See https://github.com/queone/tfe
Usage: tfe [options]
  -o [filter]              List orgs, filter option
  -m [filter]              List most recent version of modules, filter option
  -ma [filter]             List all version of modules, filter option
  -w [filter]              List workspaces, filter option
  -?, -h, --help           Print this usage page
```
